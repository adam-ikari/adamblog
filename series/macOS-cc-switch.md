# macOS cc-Switch 系列

cc-Switch 是一个用于在多个 AI 模型提供商之间快速切换的命令行工具。本系列教程帮助你在 macOS 上安装和使用 cc-Switch。

**适合谁**：macOS 用户，需要同时使用多个 AI 模型提供商
**前置要求**：macOS 12+，已安装 Claude Code
**学习目标**：掌握 cc-Switch 的安装配置和日常使用
