# 信创 + AI 工具长文配图设计

## 目标

给博客"信创系列 + AI 工具长文"补充插图，确保文字不溢出、深色/浅色主题都可读。

## 范围

约 15-18 篇概念性/分析性长文。纯代码教程、翻译系列不在此次范围（配图价值低）。

## 插图形式

- **概念图（架构/对比/流程/分类）→ 手写 SVG 矢量图**
  - 沿用 AI产业链/KylinOS 的中性配色（浅底深字）
  - 复用已有 CSS 基础设施：`img[src$=".svg"]` 白底、`img:fullscreen` 全屏锁横屏
- **界面/报错/截图类 → 网上找现成截图**
  - 存到文章同名目录
  - 优先官方文档/开源项目截图，注明来源，避免版权问题

## 优先级

### 第一批（高价值，先做）

1. 信创总览-从去IOE到自主可控 — 全景图（信创替代层 / 去 IOE 演进）
2. 国产CPU-信创六家 — 六条指令集路线对比图（LoongArch / ARM / x86 / SW64）
3. 国产数据库详解 — 六家数据库对比图（内核来源/定位）
4. 信创安全-等保与可信计算 — 等保2.0分级 + 可信计算信任根图
5. 信创供应链安全-断供与可控 — 断供/依赖链示意
6. macOS-Claude-Code-配置详解 — 配置项关系图(SVG) + 界面截图(网上找)
7. macOS-Claude-Code-报错排查 — 报错-原因-解法对照图
8. macOS-配置cc-Switch教程 — 工作流图

### 第二批（次高价值）

国产CPU-新兴势力、信创中间件与云平台、国产办公与应用软件、信创国有标准-国密OFD与测评、信创外设-打印机与其他、信创PC与整机、macOS-国产模型提供商选型指南、macOS-Claude-Code-实战案例。

## 文字不溢出保障

- SVG 中性配色（浅底深字），CSS 已加白底 + 全屏锁横屏
- 每个 SVG 自查：文字不出 rect、不出 viewBox、多行文本 rect 高度足够
- `pnpm build` 验证无报错
- 已有经验：gpu-evolution / cuda-lock-chain / domestic-gpu-vs-cuda / hbm-cowos / kylin-evolution / kylin-family-tree 的溢出修复模式

## 风格基准

AI产业链分析文章的 5 个 SVG 为风格基准：
- viewBox 宽 800，内容居中
- 标题 27px、正文 17-18px、次要 15px
- 浅色渐变背景 + 深色文字（#333/#444/#555）
- 强调色保留（绿/红/蓝/橙）但加深提高对比度
- 卡片用 `feDropShadow` 轻阴影
