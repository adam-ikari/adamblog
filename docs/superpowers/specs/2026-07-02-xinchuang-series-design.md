# 信创专题系列 设计文档

## 概述

新建一个「信创专题」系列，由 8 篇调查报告组成，覆盖信创（信息技术应用创新）全栈。每篇为调查报告类，标准同 KylinOS 报告：联网核验事实、有作者视角、书面但可读、有参考资料章节。

- 分类：信创（该分类已有 KylinOS 报告，本系列为该分类主体）
- 系列文件：posts/ 下 8 篇 + series/series-xinchuang.md 索引页
- 现有 KylinOS 报告并入本系列（作为第 4 篇「国产操作系统」的核心，或互链）

## 系列构成（8 篇）

## 写作标准（同 KylinOS 报告）

- **有作者视角**：不是中性百科，观点融在字里行间，讲透有主次，客观且观点。
- **书面但可读**：现代汉语，不用文言腔（避免"须/亦/皆"连用），不口语化。
- **联网核验**：每篇涉及的产品、架构、厂商、市场份额等事实须联网核验；核不到的标注"以官方为准"，不编造。
- **参考资料章节**：每篇末尾列出来源。
- **frontmatter**：category: 信创，tags 含信创与具体领域，series 块串好。
- **去 AI 感**：删凑数列表、空心过渡句，破三段式排比。

## 执行方式

- 8 篇工作量大，用并行 subagent 加速（每 agent 2-3 篇），每个 agent 严格按 blog-writing skill + KylinOS 报告样本执行。
- 联网核验用 WebFetch / browser-research skill。
- 第 4 篇（操作系统）复用现有 KylinOS 报告，改为系列内一篇并串链接。
- 全部完成后建 series/series-xinchuang.md 索引页，串好 prev/next。

## 诚实底线

- 不编造产品规格、性能数据、市场份额。
- 厂商宣传话术剥离，只取可核验事实。
- 各厂商产品更新快，标注"截至查阅时"。

## 不做（YAGNI）

- 不逐产品罗列参数表（选代表性产品讲透）。
- 不做购买推荐（调查报告，非导购）。
- 不重复 KylinOS 报告已详述的内容（OS 篇聚焦与麒麟的对比，不重写麒麟）。
