# 国产 CPU 新兴势力：信创之外的香山、此芯、瑞芯微、全志与平头哥

## 前言

讲国产 CPU，绕不开信创那六家，龙芯、鲲鹏、飞腾、兆芯、海光、申威，都进了信创采购的盘子，要么国资控股、要么自研指令集，能在党政军和关键行业替换里当主力（详见 [国产 CPU 详解：信创六家](/posts/国产CPU-信创六家)）。但国产 CPU 不止这六家，还有一批走别的路子的，它们未必进得了信创名录，生态和定位跟前面六家也不是一回事，可论国产 CPU 这盘棋，绕不开它们。

这篇就专讲信创之外的新兴势力：香山（开源 RISC-V）、此芯科技（ARM PC）、瑞芯微和全志（消费电子 SoC）、平头哥（阿里背景的 RISC-V 与 ARM 双线）。

先把话说清楚：这里列的几家，多数走消费电子或开源路线，不冲着信创采购去，自主程度的衡量尺度和信创六家也不同。六家信创 CPU 比的是"能不能在断供后撑起关键行业替换"，这几家比的是"能不能在各自赛道活下来、把国产 CPU 的覆盖面铺宽"。是两个维度的事，别拿同一把尺子量。

## 一、香山：开源 RISC-V 的高性能尝试

香山（XiangShan）是这批里路子最特别的一个，因为它根本不是某家公司的产品，而是一个开源项目。

它的开发者是中科院计算所（ICT）、北京开源芯片研究院和鹏城实验室，源码托管在 GitHub 的 OpenXiangShan/XiangShan 仓库，用木兰宽松许可证（Mulan PSL v2）开源，拿 Chisel 硬件描述语言写的[^xiangshan_repo]。官网 xiangshan.cc 直接打出的旗号是"目前全球性能领先的开源处理器核"[^xiangshan_site]。指令集是 RISC-V，支持 RV64GCBVH——对，就是那个 Anyone 都能免费用、不卡脖子的开源指令集。

香山的微架构按"湖"命名，已经走过三代：雁栖湖（2020 年 6 月起开发，第一代稳定版）、南湖（第二代），现在主分支上迭代的是昆明湖。2022 年它在 MICRO 这个体系结构顶会上发了篇论文，讲用敏捷方法做高性能 RISC-V 处理器，拿了三个 artifact 徽章（[来源](https://github.com/OpenXiangShan/XiangShan)）。

香山和信创六家最不一样的点在于：它不开公司卖芯片冲政企采购，而是把设计开源出来，当微架构的学术研究平台，也让工业界伙伴拿去改造成商用处理器。它的价值不在"信创名录里多一个选项"，而在两件事，一是给 RISC-V 这个不卡脖子的开源生态攒一个高性能的国产核心，二是给国内处理器微架构研究留一块能动手试验的地。这条路能不能走出能跑量的商用芯片，还得看下游愿意不愿意接，但方向上它是国产 CPU 里最"开放"的一支。

值得一提的是，论"不卡脖子"的纯粹程度，香山甚至高过龙芯，龙芯的 LoongArch 虽是自研指令集，到底还是自己一家的生态；RISC-V 本身就是开源指令集，谁都能用。代价是香山目前没有现成的商用产品链，是研究而非商品。两难全。

## 二、此芯科技：ARM 路线的 PC 级芯片

此芯科技（英文名 CIX Tech，官方域名 cixtech.com）走的路子跟鲲鹏飞腾同属 ARM 路线，但定位完全不同，它冲的是 PC，而且是 ARM 架构的 PC[^cix_site]。

公司 2021 年成立于上海，创始人兼 CEO 孙文剑，此前是 AMD 中国大业务块的负责人。注意这个出身：不是中科院、不是国防科大体系，是商业芯片公司的老兵出来创业。它做的是基于 ARM 架构的高性能通用 CPU，自研通用异构 SoC，主打高性能和低功耗平衡。

它的代表产品是 P1 芯片，官方定位"国产新一代 AI PC 算力底座"——12 核 CPU 加 10 核 GPU 再加 30 TOPS 算力的 NPU，支持 PCIe Gen4 扩展[^cix_news]。这配置摆明了冲 AI PC 去，不是冲服务器替换去的。它还盯着 Windows on Arm 的 PC 标准化，背后有联想系的资本（联想创投、联想百应在友好链接里）。

此芯科技和信创六家的差异要说清：六家信创 CPU 的 ARM 派（鲲鹏、飞腾）主攻服务器和政企，此芯科技主攻消费级 PC。它不靠政企采购活，靠的是 ARM PC 这个消费市场能不能起来。而 ARM PC 能不能起来，又得看 Windows on Arm 的生态和苹果 M 系列带起的这股 ARM 笔电风潮能吹多远。它能不能成，取决于市场，不完全取决于自主可控政策——这是它和信创 CPU 最根本的不同。

## 三、瑞芯微：消费电子的 ARM SoC 老兵

瑞芯微（Rockchip）是这批里最"接地气"的一家，做的不是通用 CPU，是 ARM SoC，主战场是消费电子和嵌入式[^rockchip_wiki]。

公司 2001 年成立于福州，fabless 模式，已经在上海证券交易所上市（603893）。它的 SoC 用 ARM 公版核心拼，覆盖平板、电视盒、Chromebook、AI 音视觉设备、IoT 硬件这些。旗舰是 RK3588——4 个 Cortex-A76 加 4 个 Cortex-A55，配 Mali-G610 GPU 和 6 TOPS 算力的 NPU，8nm 工艺，能解码 8K 视频。上一代旗舰 RK3399 还曾进过三星、华硕的 Chromebook。

瑞芯微跟信创六家的差异是一目了然的：它不做服务器替换，不做党政办公终端，做的就是消费电子和工业嵌入式的算力芯片。它的国产化程度体现在自研 SoC 整合，但核心是 ARM 公版，自主程度论不到指令集那层。可它胜在出货量大、市场活，国产 SoC 里能在消费电子这个红海里站稳并上市的，瑞芯微是头部的几家之一。它不在信创的叙事里，但在国产芯片的实际出货版图里，分量不轻。

## 四、全志：低价消费电子的 ARM SoC

全志（Allwinner Technology）跟瑞芯微是同一条赛道上的对手，也是 ARM SoC，也是消费电子为主，但更偏低价[^allwinner_wiki]。

公司 2007 年成立于珠海，fabless，深圳证券交易所上市（300458）。它的 SoC 早年主打安卓平板和 OTT 电视盒——2012、2013 年它一度是全球安卓平板应用处理器出货量第一。产品线分 A 系列平板芯片、H 系列电视盒芯片（H3、H6 这些，冲三五十美元的低价盒子市场）、R 系列工业车规、F 系列低成本嵌入。A64 这颗还被 PINE64 这类廉价开发板用。

全志的路子和瑞芯微几乎一样：ARM 公版核心，做低价消费电子 SoC，不碰信创采购。两家在国产 SoC 这条窄赛道上贴身缠打多年。全志的特点是更低价、更下沉，把国产 ARM SoC 的成本线压得很低——这让它在中国白牌平板和盒子市场曾占过不小的份额。

这两家 SoC 厂商有个共同侧面值得提：因为芯片便宜、资料相对开放，它们在开源硬件社区里活跃度很高，全志有 linux-sunxi 社区，瑞芯微在 GitHub 上开源 BSP、维护 wiki。这些社区反过来又把国产 SoC 推到了全球极客和开发板玩家手里。这条线跟信创的政企替换是两个世界，但同样是国产芯片生态的一部分。

## 五、平头哥：阿里背景的 RISC-V 与 ARM 双线

平头哥（T-Head，平头哥半导体）跟前几家出身又不一样，它是阿里巴巴集团的半导体子公司，2018 年 9 月成立，从阿里达摩院拆出。名字"平头哥"是蜜獾的绰号，取的是"又勇又聪明"那点意思[^thead_wiki]。它的底盘还有一层：成立同年的前几个月，阿里收购了杭州中天微（C-Sky Microsystems）——一家 2001 年就成立、做 32 位嵌入式 CPU 核的老厂，平头哥的嵌入式 CPU 积底子有一部分就接自这里。

平头哥的产品线分两条，走的是两套指令集。

一条是 RISC-V。2019 年 7 月它发布玄铁 910，这是平头哥的第一颗 RISC-V 处理器核，面向 IoT 和嵌入式。到 2023 年 3 月，玄铁系列已扩到八款 RISC-V 核（[来源](https://en.wikipedia.org/wiki/T-Head)）。开源这条路它走得比香山更"商用"——玄铁是奔着出货去的，不是学术平台。有意思的是，全志（就是上一节那家）跟平头哥合作，基于玄铁系列做芯片，新兴势力之间不是各走各的，而是会咬合。另一条是 ARM。2021 年 10 月平头哥发布倚天 710，服务器芯片，5nm 工艺，基于 ARM 架构，冲的是云计算和数据中心。IEEE 2024 年 3 月的一项研究评测它为"最快的 ARM 云服务器 CPU"。

平头哥和信创六家的关系要分两面看。论背景，它背靠阿里这个互联网巨头，不算国资信创体系那一脉，进的也不是党政采购的主力盘子。但论产品，倚天 710 这种服务器芯片和鲲鹏、海光在同一个赛道上，都是 ARM 服务器、都冲数据中心替换。可以说平头哥是"信创叙事之外、但和信创主力赛道有交集"的一家：RISC-V 那条线（玄铁）跟香山一个路数，靠开源指令集铺覆盖面；ARM 那条线（倚天）跟鲲鹏一个路数，靠自研核心打服务器。两条线一起，让它在这篇新兴势力的清单里占了个挺特殊的位置，既沾开源、又沾商用服务器，是阿里这种互联网大厂下场做芯片的典型样本。

还有个旁支值得一提：2019 年 9 月平头哥联合达摩院发布含光 800，阿里自称"首颗自研 AI 芯片"，做云端 AI 推理。它严格说不是通用 CPU，但和玄铁、倚天一起，构成了平头哥"自研芯片"这条线的全景——从这个侧面也能看出，阿里做芯片不只为信创，更多是为自己云和电商的算力需求兜底。

## 六、把这五家并置看

把这几家并置，能看出国产 CPU 全景里一条常被信创叙事盖住的维度：信创采购之外的国产芯片生态，其实比六家信创 CPU 更庞杂、也更贴近普通人的日常。

香山走开源 RISC-V，是科研和开源路线的代表，论"不卡脖子"的纯粹程度甚至高过龙芯——RISC-V 本身就是开源指令集，但代价是没有现成的商用产品链。此芯科技走 ARM PC，是商业创业路线，押注 ARM PC 这个消费市场能不能起来。瑞芯微和全志走 ARM SoC，是消费电子和嵌入式路线，靠出货量和成本说话，不靠政策采购。平头哥则是个跨界样本：RISC-V 玄铁铺开源覆盖面，ARM 倚天打服务器，互联网大厂下场做芯片，开源和商用两头都沾。

它们的共同点是不进信创主力盘子，要么走开源、要么走消费市场，定位和信创六家的"关键行业替换"是两套逻辑。这意味着拿"自主可控"的政策尺子去量它们，意义不大；但拿"国产 CPU 这盘棋有多大覆盖面"的尺子去量，它们是不可缺的一角。信创 CPU 解决的是关键行业不能被人卡的问题，这几家解决的则是国产芯片在更广的市场里有没有存在感的问题。两个问题都重要，只是答案在不同的赛道上找。

国产 CPU 这盘棋，一面是六家信创主力守着关键行业的安全底线，一面是这些新兴势力把触角伸向消费电子、开源和 PC——前者保的是"不能被卡"，后者争的是"能在多大范围里站着"。两条线一起看，才看得清国产 CPU 的全貌。
