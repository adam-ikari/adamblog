## 前言

## 简介

C++ 是 Bjarne Stroustrup 在 20 世纪 80 年代初搞出来的语言，底子是 C，又把面向对象那套概念揉了进去。它是一门强类型静态语言，跑得快、能干的事多，从操作系统、游戏引擎到嵌入式设备都能见到它的身影。

它身上有几个绕不开的特点：支持类、继承、多态这些面向对象的能力；模板让一份代码能复用到不同类型上；还能直接操纵内存和硬件资源，写系统软件时这点很关键；标准库则把常用的数据结构和算法都备好了，省去重复造轮子。

不过这些都是后话。如果你现在还看不太懂上面这些名词也没关系，跟着教程走一遍，自然会有个大概的感觉。



## 需要什么准备？

1. 需要一个 Linux 系统的电脑，也可以在虚拟机中安装使用 Linux。
2. 需要一个文本编辑器，可以是操作系统自带的 gedit 或者任何其他的文本编辑器。本教程推荐使用 VSCode。
3. 会简单的使用终端输入命令。

> 如果使用的是 Mac 电脑，可以通过命令`xcode-select --install`安装必要的开发工具，包括 `g++`
>
> 如果使用的是 Windows10 以上的电脑，建议使用 `WSL2`

建议使用 Linux 学习 C++的理由。

1. **开放性**：Linux 开源，允许深入操作系统底层。
2. **开发工具**：提供 GCC、GDB、CMake 等强大工具。
3. **学习环境**：支持 Shell 脚本和文件系统管理。
4. **社区支持**：丰富的文档和活跃的社区。
5. **系统编程**：符合 POSIX 标准，适合跨平台开发。
6. **实际应用**：广泛用于服务器和嵌入式系统。

## 快速上手

学一门语言，老规矩，先从 "Hello, World!" 开始。

新建一个文件 `hello.cpp`，输入以下内容：

```cpp
#include <iostream>

int main() {
    std::cout << "Hello, World!" << std::endl;
    return 0;
}
```

> 可以使用任何文本编辑器，

在终端中输入：

```bash
g++ -o hello hello.cpp
```

> `g++` 是 GNU 编译器集合的一部分，用于编译 C++程序。它将 C++源代码转换为可执行文件。通常在 Linux 系统中会预装 GNU 编译器。可以在终端输入 `g++ -v` 查看系统是否安装了 g++，并查看 g++版本。

在终端中执行生成的可执行文件：

```bash
./hello
```

这样，你就可以看到控制台输出：`Hello, World!`。

> 输出 "Hello, World!" 是编程里一个有点年头的小传统，几行代码跑通，初学者能最快摸清"写代码、编译、运行"这条链路是怎么回事。
